== Description ==


Always get the latest Ligue1 Table using this plugin.

This plugin connects to www.ligue1classement.com to get the latest Ligue1 table. This service is offered for free.

**What's new**

* The plugin is brand new.

== Installation ==

1. You need to load/activate the curl extension. If you do't know what this means, just go on. Most servers have curl installed!
2. Download the plugin and extract its contents.
3. Upload the `ligue1-table` folder to the `/wp-content/plugins/` directory.
4. Activate `Ligue1 Table` plugin through the 'Plugins' menu in WordPress.
5. In your admin console, go to Appeareance > Widgets, drag the `Ligue1 Table` widget to wherever you want it to

That's it!
